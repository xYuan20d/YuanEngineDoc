# 🛠️ 动手试试：实战案例教程

理论只有在实践中才能真正发光。本章节专门为初学者设计，通过一系列精心策划的实战小项目，带你从零开始掌握 **YuanEngine** 的核心开发流程。

---

## 📖 学习指南

这里的每一个案例都经过精心设计，旨在解决一个具体的开发问题。在开始之前，请务必阅读以下说明：

::: warning 核心原则：循序渐进
本类别的案例采用了**环环相扣**的递进式设计。
* **逻辑连贯**：后续案例的代码或场景通常建立在前面案例的基础之上。
* **难度梯度**：从最简单的“创建物体”到复杂的“脚本逻辑”，难度会逐步提升。

如果你直接跳到后面的进阶案例，可能会因为缺少前置的场景配置或基础概念而感到困惑。因此，我们**强烈建议你按照侧边栏的顺序逐一完成**，不要轻易“跳关”哦！
:::

---

## 🚀 你将学到什么？

通过本章节的练习，你将完整走一遍游戏开发的生命周期：

1. **基础场景搭建**：学会操作编辑器、管理层级结构。
2. **物理系统应用**：掌握刚体、碰撞体以及物理材质的配合。
3. **脚本逻辑实战**：学习如何使用 JavaScript API 控制游戏对象的行为。
4. **综合案例演练**：将 UI、音效、物理和脚本整合在一起，构建一个真正的游戏原型。
